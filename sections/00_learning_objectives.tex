\subsection*{Learning objectives}
\noindent
In this tutorial,
I'll show you how to:

\begin{itemize}
	\item \textbf{use} the notation for limits, derivatives, and integrals
	\item \textbf{evaluate} limit expressions
	\item \textbf{calculate} derivatives of functions
	\item \textbf{solve} optimization problems
	\item \textbf{compute} integrals geometrically as areas
	\item \textbf{compute} integrals symbolically using pen-and-paper %  calculation and SymPy
	\item \textbf{compute} integrals numerically % using \tt{scipy.integrate.quad}
	\item \textbf{understand} the inverse relationship between derivatives and integrals
		in the fundamental theorem of calculus
	\item \textbf{determine} if a series converges or diverges
	\item \textbf{calculate} Taylor series approximations for functions
\end{itemize}
